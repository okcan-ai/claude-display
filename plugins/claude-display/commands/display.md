# /display

Open the Claude Display dashboard and show a welcome message.

## Instructions

1. Call the `open_dashboard` tool to open the browser dashboard
2. Call the `display` tool with a welcome message in markdown format
3. Tell the user the dashboard is ready and what tools are available

Example welcome message:
```markdown
# Welcome to Claude Display

Your visual companion dashboard is ready. I can show you:
- **Rich content**: HTML, Markdown, images, code
- **Charts & tables**: Data visualizations
- **Panels**: Persistent sidebar information
- **Notifications**: Toast alerts

Ask me to display something!
```
